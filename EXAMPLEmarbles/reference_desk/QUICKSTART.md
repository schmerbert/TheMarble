# QUICKSTART.md

Use this when arriving cold or when work is moving fast.

## One-minute arrival

1. Name the active matter.
2. Check `HANDOFF.md` for live state.
3. Check `RED_FLAGS.md` for warnings.
4. Check `RESTRICTED_STACKS.md` for silence rules.
5. Decide whether the task is intake, search, drafting, stamping, vault packet, receipt, or handoff.
6. Use a quick slip from `DESK_SLIPS.md`.

If changing matters, use Quick Matter Switch before returning any stored context.

If no matter exists yet, use Quick Matter Open.

## Nine common moves

### Open a matter

Use Quick Matter Open.

Create or update `MATTER_MAP.md`.

Do not import facts, deadlines, or authorities from another matter unless cross-matter return is explicit and restrictions are checked.

### Receive a document

Use Quick Intake Slip, then `TRIAGE.md`.

Land in `INTAKE_CLEARING.md`.

If the document creates leads, add Quick Wild Lead slips to `WILD_STACKS.md`.

Triage may create Wild leads and Warnings. It does not create Ground.

### Surface useful material

Use Quick Search Return.

Return Ground only from stamped records. Return Wild results as Pressure or Scent.

### Capture a possible fact

Use Quick Wild Lead first.

Only Stamp into `MATTER_FILE.md` if source, attribution, status, and use limit are clear.

### Capture possible law

Use Quick Wild Lead first.

Only Stamp into `AUTHORITY_SHELF.md` if citation, jurisdiction, force, currentness/status, source, and use limit are clear.

### Capture a possible deadline

Use Quick Red Flag if uncertain.

Only Stamp into `PROCEDURE_CALENDAR.md` when source rule/order, calculation basis, forum, and review status are clear.

### Prepare work for the vault

Use Quick Vault Packet.

Land in `VAULT_DOOR.md`. Do not claim it was filed, saved, sent, or approved.

### Record what happened in the vault

Use Quick Receipt.

Land in `RECEIPT_SLOT.md`. A receipt is Ground only for the event it reports.

### Start or end the day

Use Quick Daily Desk Card or `DAILY_DESK.md`.

Warnings surface before interesting leads.

## Stop signs

Stop and label Warning if:

- matter is unclear
- source is missing
- a legal conclusion is model-only
- a deadline is not sourced
- currentness/status is unknown
- restricted material may return too broadly
- a vault event is claimed without receipt
- work from another matter appears

## Desk sentence

When unsure, say:

```text
I can hold this as a Wild lead, but I cannot let it count until it is stamped.
```
