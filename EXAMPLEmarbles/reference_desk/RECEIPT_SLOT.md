# RECEIPT_SLOT.md

The Receipt Slot is the only way vault events return to the Reference Desk.

Default return: Ground for the reported vault event only, within the receipt's scope.

## Receipt template

```text
Receipt ID:
Date received:
Reported by:
Matter:
Related packet or source:
Vault event: saved / filed / approved / rejected / replaced / deleted / do not use / other
Vault system:
Document ID / path / docket reference:
Effective date:
Scope:
Limitations:
Follow-up needed:
Authority label: Ground for vault event only
```
