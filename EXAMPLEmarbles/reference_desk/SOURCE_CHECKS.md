# SOURCE_CHECKS.md

Source Checks record attempts to verify whether a source, matter, citation, rule, docket, or document reference can be found.

Default return: Pressure or Warning.

A failed lookup is not proof that something does not exist. It is a record that a specific search was attempted and did not find a match.

## Source check template

```text
Check ID:
Date:
Matter:
Target:
Search method:
Queries:
Result:
What this proves:
What this does not prove:
Next check:
Authority label:
```
