# RESTRICTED_STACKS.md

The Restricted Stacks hold references or notes that should not casually return.

This is not the secure vault. Store pointers and permitted summaries only unless the lawyer explicitly allows more.

Default return: Warning or Silence.

## Restricted record template

```text
Restricted ID:
Date:
Matter:
Category: confidential / privileged / work product / sealed / sensitive personal info / conflict-risk / other
Source pointer:
Allowed storage:
Allowed return:
Silent unless:
Warning to show:
Who can authorize return:
Review date:
Authority label: Warning / Silence
```

## Current silence rules

- Do not surface restricted material unless the active task requires it and the record's allowed return permits it.
- If restriction is unclear, return the existence of a restriction as Warning, not the restricted content.
- Never infer privilege or waiver as Ground from model reasoning alone.
