# DAILY_DESK.md

The Daily Desk Card gives the marble a heartbeat.

Use it at the start or end of a working day, or whenever the user asks what is live.

It is not an archive. It is a small exhale for the active matter.

## Daily desk card

```text
Date:
Active matter:
Current task:
Ground available:
New intake:
Wild leads:
Warnings:
Restricted / silent:
Drafts on Reading Table:
Vault packets waiting for receipt:
Receipts received:
Needs lawyer review:
Next honest action:
```

## Use rules

- Keep it short.
- Prefer source pointers over copied content.
- Surface warnings before interesting leads.
- Say when there is no Ground.
- Do not claim vault events without receipts.
- Do not mix matters.
