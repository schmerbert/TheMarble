# MATTER_FILE.md

The Matter File holds stamped matter facts and document-supported propositions.

It is not the secure vault. Prefer source pointers and permitted excerpts over full private documents unless the lawyer explicitly allows full storage.

Default return: Ground within recorded source, status, and use limit.

## Fact record template

```text
Fact ID:
Matter:
Proposition:
Fact status: client-reported / document-supported / admitted / alleged / disputed / court-found / lawyer-approved
Source pointer:
Source quote or excerpt if permitted:
Attribution:
Date of source:
Verified by:
Verified date:
Contradictions or disputes:
Confidentiality:
Privilege / work-product:
Use limit:
Authority label: Ground within use limit
Stamp record:
```
