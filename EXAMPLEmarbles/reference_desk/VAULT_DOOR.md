# VAULT_DOOR.md

The Vault Door is the boundary between the Reference Desk and the lawyer's secure system.

The model does not enter the vault.

Beyond the Vault Door is not memory.

## Outgoing packet rule

The desk may prepare a packet for lawyer review or vault filing. The packet is not filed, saved, approved, sent, or relied on unless a receipt later says so.

## Outgoing packet template

```text

Contains confidential info: yes / no / unknown
Contains privileged or work-product material: yes / no / unknown
Authorities verified: yes / no / partial
Facts verified: yes / no / partial
Source list:
Required lawyer review:
Do-not-use warnings:
Receipt needed:
Authority label: Path
```
