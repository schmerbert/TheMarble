# Ledger

Tended ground. Stand on it.

## Entry requires

- **Verbatim extraction:** quote or file span + source path + date
- **User confirmation:** explicit yes to a decision
- **Attributed synthesis:** your insight + `source:` pointing at a Log entry or file

This crossing is named **Promote**. If a candidate entry does not satisfy one requirement above, do not Promote it.

## Frontmatter for new entries

```yaml
---
author: user | instance
source: path or quote reference
date: YYYY-MM-DD
voiced: true | false
---
```

## Refused

- Unsigned model inference
- Summary without source
- Log content copied without Promote
