## One line each

- **Bench** — The Honda Civic. Unsigned inference never enters the Ledger. Markdown-only; the gate is convention until code arrives.
- **Trench** — Do not lift from blind soil. Archaeology duty station; the Lift gate refuses in JavaScript.
- **Fall** — Mist, not creature. Water washes in; you Settle what earns depth; downstream gets a labeled stream. Flagship breath router.
- **Reference desk** — Lawyer-directed legal continuity at the edge of the Wild Stacks. The vault door stays the lawyer's; the model never certifies.
- **Cabin** — Forest continuity engine. Extract never summarize; Wild → Home only by attributed synthesis. Home marble that grew up.
- **Lighthouse** — The light must not lie. Dead reckoning is legal while labeled; the poison is only ever the missing label. Ships / Keeper pull pair.

## Cross-model success

Six marbles, **five different builders** — and each one landed an operable first pass:

| Builder | Marbles | What they tended toward |
|---|---|---|
| **Cursor** | Bench, Fall | Fast iteration, large surfaces, cold-judge demos |
| **Codex** | Trench, Reference desk | Sharp mechanism (one door) and domain depth |
| **Claude Code** | Cabin | Long-horizon engine work, hooks, hostile floor at scale |
| **Claude Fable 5** | Lighthouse | Epistemic hygiene, breath layer, structural wall tests |

That variety matters. The manual is not "how to prompt one model to make a pretty folder." It is closer to: **if you can name the cost and the forbidden cell, a cold instance on almost any serious agent can build something inheritable.** The specimens are the evidence — not hype about any one model, but six different workers leaving six different places that actually work.

They did not all one-shot perfection. Fall needed inside passes. Lighthouse caught stale handoffs on its first live `breathe out`. Cabin's handoff still flags open threads. Reference desk is mock-triage only. What held across builders was the **same bones**:

- a named forbidden cell
- a real crossing (or an honest admission that enforcement is still convention)
- at least one hostile test on the dangerous path
- a handoff rewritten at session end

Different aesthetics. Same law. **The framework traveled; the marbles did not collapse when the builder changed.**
