# Hostile test

Name: **lifted find with no context cut cannot enter the Context Register**

The test attempts to write a lifted artifact without layer, grid, coordinates, relationship, or crossing.

Pass means the Lift Gate refuses the write and no Context Register file appears.

Second hostile name: **identification does not outrank context**

The test attempts to write a lifted "possible coin" using identification as evidence, but without layer, grid, coordinates, or relationship.

Pass means the Lift Gate refuses the write. "Coin" is not context.

Third hostile name: **clean Lift requires transfer**

The test attempts a lifted record with context fields but without in-situ status, photo, label, or destination.

Pass means the Lift Gate refuses the write. A crossing is not clean unless the physical transfer is recorded.

Positive living-state test: **visible not ready enters Field Notebook**

The test writes an embedded visible find as `--state observed`.

Pass means the visible state can be preserved without pretending it is ready to lift.
